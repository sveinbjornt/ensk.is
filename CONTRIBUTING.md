# Contributing to ensk.is

Thank you for your interest in contributing to the [ensk.is](https://ensk.is) English-Icelandic dictionary project. We welcome corrections, new words, and improvements to the codebase.

## How to Contribute

1. **Report Issues:** Use the GitHub issue tracker to report errors, missing words, or bugs.
2. **Pull Requests:** Submit pull requests for code changes or dictionary additions.
    * For dictionary additions, please ensure they follow the established format.
